# GLPI Telegram Bot

Automação completa de chamados do GLPI via Telegram utilizando n8n.

## Visão Geral

Projeto criado para facilitar o atendimento técnico e agilizar operações do Service Desk diretamente pelo Telegram, sem necessidade de acessar o painel web do GLPI para tarefas rotineiras.

## Funcionalidades

- Abrir chamados pelo Telegram
- Consultar chamados por número
- Alterar status de chamados
- Fechar chamados com solução
- Adicionar comentários
- Enviar fotos e vídeos para chamados
- Fluxo Foto/Vídeo + Solucionar
- Menus interativos para usuários
- Sessões inteligentes de atendimento

## Tecnologias Utilizadas

- n8n
- GLPI API REST
- Telegram Bot API
- JSON Workflows
- Automação de Processos

## Benefícios

- Redução de tempo operacional
- Atendimento mais rápido
- Menos acessos manuais ao GLPI
- Maior produtividade da equipe técnica
- Melhor experiência para usuários internos
